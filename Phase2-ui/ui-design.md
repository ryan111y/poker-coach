### 2.0B 对局界面

```
┌─────────────────────────────────────┐
│  ← 返回    VS Tom Dwan   手牌 #3   │
├─────────────────────────────────────┤
│                                      │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐│
│  │ K♦ │ │ 7♠ │ │ 2♣ │ │   │ │   │ │
│  │ 翻  │ │ 翻  │ │ 翻  │ │转  │ │河  ││
│  └────┘ └────┘ └────┘ └────┘ └────┘│
│                                      │
│  ┌─────────┐    ┌─────────┐         │
│  │ Tom     │    │ 你       │         │
│  │ Dwan    │    │         │         │
│  │ 过牌    │    │ ？      │ ← 到你了 │
│  └─────────┘    └─────────┘         │
│                                      │
│  底池：45BB         你的手牌：A♥K♥    │
│  实时胜率：72%                        │
│                                      │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐       │
│  │弃牌│ │过牌│ │下注│ │全下│       │
│  └────┘ └────┘ └────┘ └────┘       │
│                                      │
│      Tom Dwan 正在思考...  ⏳       │
└─────────────────────────────────────┘

**布局要点：**
- 顶部栏：返回 + 对手名 + 手牌轮次
- 公牌区：5 张牌位，按阶段显示
- 行动区：左右布局（对手 vs 你）
- 底池/手牌/胜率：信息行
- 操作按钮：弃牌/过牌/下注/全下，四个按钮
- AI 思考时显示加载动画 + 模拟思考时间（2-4秒）
- 按钮状态：不可行的行动灰显（如筹码不够全下）

**操作流程：**
1. AI 先行动（显示思考动画）
2. AI 出牌后 → 你选择行动
3. 轮到 AI → 重复
4. 摊牌或有人弃牌 → 进入复盘
```

## 三、组件设计

### 3.1 扑克牌组件

**选牌状态：**
```
┌──────┐   ┌──────┐   ┌──────┐
│  A♥  │   │  K♥  │   │  ＋  │
│      │   │      │   │     │
│  ♥   │   │  ♥   │   │     │
└──────┘   └──────┘   └──────┘
 已选红      已选红      未选

尺寸：52 × 72px（触摸目标）
边框：已选 → 2px #e94560 实线
      未选 → 2px #3a3a5e 虚线
花色：♥♦ → #e94560  /  ♠♣ → #1a1a2e（背景上显示黑）
悬停：rgba(233,69,96,0.1) 背景
```

**公牌状态：**
```
┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐
│ K♦ │ │ 7♠ │ │ 2♣ │ │ 9♥ │ │ 3♠ │

 翻牌绿     翻牌绿    翻牌绿   转牌金   河牌红

尺寸：46 × 64px（比手牌小，区别层级）
边框颜色按阶段：翻牌 #16c79a / 转牌 #f0a500 / 河牌 #e94560
```

**选牌弹出层：**
```
┌──────────────────────┐
│         ♥           │
│  [A][K][Q][J][10][9] │
│  [8][7][6][5][4][3]  │
│  [2]                │
│                      │
│         ♠           │
│  [A][K][Q][J][10][9] │
│  [8][7][6][5][4][3]  │

│                      │
│  ♦  +  ♣  （切换）  │
│                      │

### 2.2 牌局时间线组件

```
 翻牌前        翻牌          转牌         河牌  
 └─────┼───────────┼────────────┼──────────┼───→
  [跟注3]    [下注20✓]     [过牌]      [弃牌]
                 ↑
           点击这里试试过牌

交互：
- 水平滚动时间线
- 已完成决策点 → 灰色圆角标签
- 关键决策点 → 红色边框 + 悬停效果（← 用户实际做过的选择）
- 点击关键决策点 → 弹出替代方案
- 点击其他决策点 → 无反应（灰色表示已发生不可改）
```

## 四、动效规范

```
按钮点击：    0.1s 缩放效果
