# 🃏 德扑复盘教练

> 分析你的每一手牌，看看"如果换一种打法"会怎样

## 项目结构

```
poker-coach/
├── Phase0-design/     # 交互设计 · 线框图 (SVG)
├── Phase1-tech/       # 技术方案 · 开发路线
├── Phase2-frontend/   # 前端代码 (开发中)
├── Phase3-backend/    # 后端代码 (开发中)
└── README.md
```

## 产品核心

MVP：手动输入一手牌 → 选择一个决策点 → 
Monte Carlo 模拟 → 对比分析 → LLM 复盘

## 技术栈

前端: React + Tailwind CSS
后端: Python + FastAPI
引擎: Python (treys + Monte Carlo)
数据库: PostgreSQL/SQLite
部署: Docker
