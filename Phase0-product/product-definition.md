# Phase 0: 产品定义

> 德扑复盘教练 · 从"数字复盘"到"平行宇宙复盘"

---

## 一、痛点分析

### 核心痛点
德扑玩家打完一手牌，最常问的问题是：
- _"我当时该不该跟？"_
- _"如果我 bluff 会不会拿下？"_
- _"翻牌 check 而不是下注会怎样？"_

**问题是：没有人能回答。**
- 自己复盘：记不住细节，靠感觉猜
- 和朋友聊：朋友看到的也是主观判断
- 现有工具：给你一堆数字，你要自己读

### 用户场景
```
场景A：晚上打了一手牌，AK翻牌中顶对，你下注被跟注。
转牌出花面，你过牌，对手下注，你弃牌。
躺下睡不着——"他是不是在 bluff？"
"如果我翻牌 check 引诱他诈唬，是不是能多赢？"

场景B：在 2/5 局上输给了牌友老张的一手大牌。
老张是个非常紧的玩家，只拿好牌才出手。
"如果我翻牌 heavy bet，老张会不会 fold？"
"如果对手是 Tom Dwan 那种疯子，他会怎么打？"
```

### 现有方案的局限
| 方案 | 局限 |
|------|------|
| 自己回忆 | 记不住细节，选择偏误 |
| 问朋友 | 朋友也是主观判断 |
| PokerTracker | 只记录你做了什么，不模拟如果做了别的会怎样 |
| Equilab | 只能算摊牌胜率，不能模拟整条街的决策 |
| GTO Wizard | 太复杂，中低水平玩家看不懂 |
| 问 ChatGPT | 会给你一个"听起来对"的答案，但背后没有真实的模拟 |

### 我们的回答
**"平行宇宙复盘"** — 在实际发生的牌局中，选一个关键决策点，系统跑 Monte Carlo 模拟告诉你：如果当时选了另一个打法，结果是怎样的。

**独家杀招：AI 牌手模拟** — 不仅可以模拟"默认"对手，还可以：
1. 模拟你认识的牌友（创建他们的打法特征）
2. 模拟职业牌手（Tom Dwan、谭轩、Phil Ivey 等名将）
3. 系统根据打牌风格参数（VPIP、PFR、激进度、位置倾向等）生成 AI 对手

不是猜，是跑了 10000 次模拟的真实数据。
不是打空气对手，是模拟你真实面对的玩家。

## 三、竞品分析

### 直接竞品

| 维度 | PokerTracker 4 | Equilab | GTO Wizard | 我们要做的 |
|------|---------------|---------|------------|-----------|
| 价格 | $99 买断 | 免费/$49 Pro | $35/月 | 免费基础/付费高级 |
| 核心功能 | 追踪战绩 | 算胜率 | GTO 策略 | 打法模拟对比 |
| 能不能模拟"换打法" | ❌ | ❌ | ❌ | ✅ |
| 门槛 | 中高 | 中 | 高 | 低 |
| 是否说人话 | ❌ 纯数字 | ❌ 纯数字 | ❌ 纯策略 | ✅ LLM 总结 |
| 移动端友好 | ❌ 桌面端 | ❌ 桌面端 | ✅ Web | ✅ Web |

### 间接竞品

| 产品 | 能做的 | 不能做的 |
|------|--------|----------|
| ChatGPT | 聊牌、给建议 | 背后没有真实模拟，全靠编 |
| 德州扑克论坛 | 问别人的经验 | 别人不一定对，也不一定回复 |
| 教练/培训 | 教你怎么打 | 贵（500+/小时），不能随时用 |

### 我们的差异优势
1. **模拟"平行宇宙"** — 不是教你"应该怎么做"，而是告诉你"如果做了会怎样"
2. **AI 牌手模拟独家** — 模拟真实的牌友和职业牌手，不是普适的"紧/中/松"
3. **真实数据** — 不是 AI 编的，是货真价实的 Monte Carlo 模拟跑出来的
4. **门槛低** — 不用懂 GTO 也能用，点了就出结果
5. **两个版本都行** — 初级玩家看结论，高级玩家看数据

## 四、功能优先级

### 优先级分级说明
```
P0 — 不做就产品不成立
P1 — 重要，但没有也能用
P2 — 锦上添花
P3 — 以后再说
```

### P0：MVP 必须做
```
□ 手牌输入
   - 选自己的手牌（点选，不靠打字）
   - 选公牌
   - 填每轮行动和底池
   - 设对手人数和位置

□ 决策点选择
   - 牌局时间线展示
   - 点击已做决策，弹出替代方案
   - 支持替代：过牌/下注/加注/弃牌
   
□ AI 牌手模拟（⭐ 独家核心）
   - 预设职业选手资料库：
     · Tom Dwan（激进、大诈唬）
     · Phil Ivey（全面、读牌强）
     · 谭轩（华人职业选手）
     · Daniel Negreanu（读牌流）
     · 更多可后续添加
   - 自定义牌友：用户创建自己的牌友档案
     · 方式A：输入几手跟他打过的牌 → 系统自动提取风格参数
     · 方式B：自然语言描述 → LLM 生成风格参数
     · 方式C：手动调整参数滑块
   - 🧬 AI 牌手进化机制（最核心的杀手功能）：
     · 每次你用这个牌友复盘，系统自动记录他的打法
     · 手数越多 → 风格参数越精确 → 模拟越像真人
     · 从新手打成老油条——100手后跟打真人没区别
     · 参考：20手 ≈ 初步了解，50手 ≈ 很准，100手+ ≈ 克隆
   - 系统根据牌手档案 + 累计数据模拟对手的打法决策
   
□ Monte Carlo 模拟
   - 牌力评估（treys 引擎）
   - AI 牌手行为决策模块
   - 10000次迭代
   - 模拟时长 <5秒

□ 结果展示
   - EV 对比（实际 vs 替代）
   - 胜率对比
   - 差异值显示
   - 胜负分布图表

□ 核心交互
   - 加载中状态
   - 错误提示
   - 响应式（移动+桌面）
```

### P1：下个版本做
```
□ 牌谱历史记录
   - 保存输入过的牌局
   - 历史记录列表
   - 重新打开历史牌局

□ 牌手资料库扩充
   - 更多职业选手数据
   - 用户共享自定义牌友
   - 牌手排行榜/热门牌手
   
□ LLM 复盘总结
   - 调用大模型生成文字复盘
   - "对比 Tom Dwan 的风格，他在这手牌有 70% 概率会 bluff..."
   
□ 对手范围自定义（手牌组合级）
   
□ 多决策点对比
   - 同一个牌局对比多个决策点
   - 决策点间的关联分析
```

### P2：有了更好
```
□ 牌谱导入
   - 文字牌谱复制粘贴
   - 图文识别拍照导入
   - PokerTracker 数据导入

□ 用户体系
   - 注册/登录
   - 数据云端同步

□ 多语言
   - 英文版（国外用户能赚钱）
```

### P3：未来想象
```
□ 在线对局实时分析
□ 跟真人牌谱库对标
□ AI 教练模式（全程指导）
□ 付费墙（高级功能）
```

## 五、成功指标

### MVP 验证指标
```
□ 用户完成一次完整复盘 < 3 分钟
□ 用户看了结果后说"原来如此"或"学到了"
□ 周留存率 > 30%
□ 用户愿意分享给牌友
```

### 商业指标
```
月活用户 > 1000  → 验证需求
付费转化 > 5%    → 验证价值
复购率 > 60%     → 验证黏性
```

## 六、产品边界（不做什么）

### MVP 明确不做的
```
❌ 不追踪你的战绩（跟 PokerTracker 不竞争）
❌ 不做在线对战
❌ 不教你从头学德扑
❌ 不做视频/直播
❌ 不做社区/论坛
❌ 不做移动端 App（先做 Web）

重点是：把一个核心功能做到极致，不是堆功能。
```

## 六B、商业模式猜想

### 免费功能
- 基础模拟：紧/中/松 对手，1 个决策点对比
- 预设职业选手库（Tom Dwan、谭轩等）

### 付费功能
- 自定义牌友创建（从牌局/描述生成）
- 🧬 AI 牌手进化（记录+更新牌友数据）
- 无限制模拟次数
- 多决策点同时对比
- 牌友数据导出

### 独特的转化逻辑
用户创建了
